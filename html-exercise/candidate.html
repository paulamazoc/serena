S&L Candidate Assessment

These questions use the JSON in candidate.json and the HTML skeleton below

Using the sample JSON in candidate.json and the HTML skeleton below, write JavaScript (in candidate.js) that could accomplish the following:

  • Populate the lists with all of the products, grouped by category, displaying their relevant information in the format indicated. If the product is available at more than one store, the stores should be listed separated by ', '.
  • The links should point to: 'http://www.serenaandlily.com/products/theIdOfTheProduct'.
  • Populate the select menus with the relevant options. Selecting the options should hide and reveal products as appropriate.
  • Selecting the LOCATION or CATEGORY option should clear the filter for that attribute.
  • If there are no products that match the selected filter in a given category, the category header should be hidden as well, and revealed again when the filter is cleared.

You may incorporate jQuery if you wish, or use vanilla JS.
